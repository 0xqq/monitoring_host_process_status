## 远程主机及进程监控系统（version1.0）
1. 实时监测主机及所关注进程的状态
2. 出现异常（停止）向用户提供预警信息，发送邮件
3. 用户和主机账号信息加密
4. 提供控制功能（运行、停止）
5. 提供操作界面，最大的方便和简洁提供给用户使用


### 目录介绍
* client_monitoring  用户Web客户端，用于查看待检测的进程
* server_monitoring  监测服务器端，用于定时检测进程状态,并将检测结果存入MySQL数据库中
